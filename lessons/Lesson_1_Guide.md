# Lesson 1: Introduction to Luna's POS System

## Overview
Luna's POS is a web-based Point of Sale system built for **Luna's Arrozcaldo**, a classic Ilonggo restaurant with branches across Iloilo City.

## What You Will Learn
- What a POS system is and why it matters
- The purpose and features of Luna's POS
- How to navigate the system as a first-time user

---

## 1.1 What is a POS System?
A **Point of Sale (POS)** system is a combination of software and hardware that allows businesses to process sales transactions. It replaces the traditional cash register and helps manage orders, inventory, and customer data in one place.

## 1.2 About Luna's POS
Luna's POS was created to digitize and streamline operations at Luna's Arrozcaldo. It supports:

- **8 branches** across Iloilo City
- Role-based access for **admins and staff**
- Real-time sales tracking and reporting

## 1.3 Key Features
| Feature | Description |
|---|---|
| POS Terminal | Fast, cashier-friendly order and checkout interface |
| Dashboard | Real-time KPIs: revenue, orders, and top products |
| Inventory | Track stock levels and receive low-stock alerts |
| Customers | Manage customer records and order history |
| Sales Reports | View revenue trends and order source breakdowns |
| Authentication | Secure login with Google Sign-In support |
| PWA | Installable as a mobile or desktop app |

## 1.4 Tech Stack
- **Backend:** PHP
- **Database:** PostgreSQL (Supabase-compatible)
- **Frontend:** Vanilla HTML, CSS, JavaScript

---

## Summary
Luna's POS is a complete restaurant management system that handles sales, inventory, customers, and reporting — all in one web application accessible from any device.
